# UC3 – Accept User Slot Input (1–9)

## Goal
Allow the user to enter a slot number between 1 and 9.

## Actor
Human Player

## Flow
1. User is prompted to enter slot number
2. Program reads integer input
3. Input value is returned to game logic

## Key Concepts Used

### 1. User Input Handling
- Scanner class is used to take input from user

### 2. Method Creation
- A separate method `getUserInput()` is created

### 3. Separation of Concerns
- Input logic is separated from game logic

## Key Requirements
- Read integer input
- Return the slot value

## Output Example
Player 1 (X), enter slot (1-9): 5
